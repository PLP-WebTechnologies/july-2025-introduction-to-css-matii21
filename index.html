Understanding CSS Fundamentals

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Why CSS Matters

CSS is essential for creating visually appealing websites. It allows developers to separate content from presentation, making websites more maintainable and accessible.

  • Separation of concerns
  • Consistent design across pages
  • Easier maintenance
  • Improved accessibility

The CSS Box Model

In CSS, every element is represented as a rectangular box. The CSS Box Model describes how these boxes are structured and how their dimensions are calculated.

Content

Box Model Components

Content

The actual content of the box, where text and images appear.

Padding

Transparent area around the content inside the border.

Border

The edge around the padding (if any) and content.

Margin

Transparent area outside the border, separating elements.

Box Model in Practice

See how different box model properties affect element dimensions and spacing.

Standard Box Model (content-box)

Width and height apply only to the content area.

Width: 200px
Padding: 20px
Border: 5px

Total width: 200 + (20×2) + (5×2) = 250px

Border Box Model

Width and height include padding and border.

Width: 200px
Padding: 20px
Border: 5px

Total width: 200px (includes padding and border)
